# MVP_ORCA — TechGuard Labs Pentest Audit Platform

Complete deployable bundle of the TG Audit Orchestrator + PT-Orc scan engine.

---

## One-Command Setup (Fresh Kali Linux)

```bash
git clone https://github.com/Surya-Chinnathambi/MVP-ORCA.git
cd MVP-ORCA
bash setup.sh
```

With demo team + sample VAPT project pre-loaded:

```bash
bash setup.sh --demo
```

Then start the server:

```bash
cd tg-audit-orchestrator
source .venv/bin/activate
uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```

Open **http://localhost:8000**

## What `setup.sh` Does

| Step | Action |
|------|--------|
| 1 | Verifies Python 3.11+ |
| 2 | Installs system packages: `tesseract-ocr`, `nmap`, `dnsutils`, `curl`, `libpq-dev` |
| 3 | Installs Python jq shim → `~/.local/bin/jq` |
| 4 | Sets PT-Orc scripts executable + updates path in scan runner |
| 5 | Creates Python virtualenv + installs all pip packages |
| 6 | Generates `.env` with a random `SECRET_KEY` |
| 7 | Creates `data/` directories |
| 8 | Runs `alembic upgrade head` (creates SQLite DB + all tables) |
| 9 | Seeds 10 RBAC roles + admin user |
| 10 | *(--demo only)* Creates 4 team members + VAPT project with sample findings |

## Manual Setup (Without setup.sh)

```bash
git clone https://github.com/Surya-Chinnathambi/MVP-ORCA.git
cd MVP-ORCA

# 1. System packages
sudo apt-get install -y python3-venv tesseract-ocr nmap dnsutils curl

# 2. jq shim
cp tools/jq ~/.local/bin/jq && chmod +x ~/.local/bin/jq
export PATH="$HOME/.local/bin:$PATH"

# 3. Python env
cd tg-audit-orchestrator
python3 -m venv .venv && source .venv/bin/activate
pip install -e ".[dev]"

# 4. Config
cp .env.example .env
# Edit .env — set SECRET_KEY and CLAUDE_API_KEY

# 5. Database
mkdir -p data/{evidence,deliverables,scan_jobs}
alembic upgrade head
python scripts/seed.py

# 6. Start
uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```
